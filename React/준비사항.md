# 준비사항

1. Node.js

    - 웹팩과 바벨 같은 도구들이 Node.js 기반으로 만들어져 있다
    - 이러한 도구를 사용하기 위해서 Node.js를 설치
  
2. Yarn

    - Yarn은 개선된 버전의 npm.
    - npm은 Node.js를 설치할때 패키지 매니저 도구.
    - 더 나은 속도, 더 나은 캐싱 시스템을 사용하기 위함.
  
3. 코드 에디터: 주로 VSCode(Visual Studio Code)를 사용

4. 윈도우의 경우 Git Bash를 이용
