# React 설치 및 사용

1. 설치
    - create-react-app 는 리액트 앱을 만들어주는 도구

```vscode
npm install -g create-react-app
```


```vscode
yarn global add create-react-app
```
2. 사용방법

    - 터미널에서 명령어를 입력
 ```vscode
 create-react-app hello-react 
 ```
 
 성공적으로 설치 완료 후
 
 ```vscode
 cd hello-react
 yarn start 혹은 npm start 
 ```
 
 이후 localhost:3000에서 실행창을 확인할 수 있다.
