# 알고리즘

[백준 Main_10171_고양이](https://www.acmicpc.net/problem/10171);

```java
public class Main_10171{
	System.out.print("\\    /\\\n");
	System.out.print(" )  ( ')\n");
	System.out.print("(  /  )\n");
	System.out.print(" \\(__)|\n");
}

```
- 위의 코드와 같이 고양이 그림을 출력하는 것인데 문제는 sysout 첫 번째줄 \ 이 문자를 조심해야 한다
- 있는 그대로의 \을 출력하기 위해서는 \ \ 로 코딩을 작성
